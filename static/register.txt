vizualização de rotas no hospedes:

- /hospedes/ (GET) - lista os hospedes
- /hospedes/ (POST) - insere hospedes ao servidor
- /hospedes/new (GET) - furmulario para criar um hospede
- /hospedes/<id> (GET) - obter dados de um hospede por id
- /hospedes/<id>/edit (GET) - formulario para editar hospede
- /hospedes/<id>/update (PUT) - atualiza os dados de hospedes
- /hospedes/<id>/delete (DELETE) apaga registros de um hospede
